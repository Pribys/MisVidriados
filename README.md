# Mis Vidriados

Mis Vidriados es una aplicación técnica para la formulación, análisis y documentación de vidriados cerámicos.

El proyecto integra herramientas de cálculo químico, gestión de materias primas, base de datos de vidriados, análisis de fórmula Seger y método Ian Currie en un entorno de trabajo estructurado.

---

## Filosofía

Mis Vidriados se distribuye como software libre bajo la licencia GNU General Public License v3 (GPL-3.0).

El objetivo del proyecto es:

- Fomentar la formulación rigurosa de vidriados
- Apoyar la formación técnica en cerámica
- Proporcionar herramientas de cálculo transparentes y verificables
- Garantizar que las mejoras permanezcan accesibles para la comunidad

El software está diseñado con especial énfasis en la claridad, la reproducibilidad y la coherencia química.

---

## Módulos principales

- Teoría de vidriados
- Base de datos de vidriados
- Gestión de materias primas
- Cálculo de fórmula Seger
- Método Ian Currie
- Cuaderno de experimentación
